A modern, responsive personal portfolio website showcasing my journey as a Cloud Practitioner and Full-Stack Web Developer. Features a **light/dark mode toggle**, **Markdown-powered blog**, and fully customizable sections.



## ✨ Features

- 🎨 **Light & Dark Mode** - Seamless theme switching with localStorage persistence
- 📱 **Fully Responsive** - Optimized for all devices (mobile, tablet, desktop)
- 📝 **Markdown Blog** - Static blog posts rendered with `marked.js` (no backend needed)
- 🚀 **Pure Static** - Zero dependencies, works entirely on GitHub Pages
- 🎯 **Interactive UI** - Smooth animations, hover effects, and modals
- 📄 **Resume Download** - One-click PDF download functionality
- 🔧 **Modular Code** - Separated HTML, CSS, and JavaScript for easy maintenance

## 🛠️ Technologies Used
